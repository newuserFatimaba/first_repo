Salut, je suis étudiante en développement web

J’ai choisi d’apprendre le développement web parce que je suis passionnée par l’informatique.
En effet, j’ai toujours voulu connaitre le secret qui se cachait derrière les applications web et pour cela il me fallait apprendre les langages de programmation. C’est en cela que je me suis lancée dans le domaine informatique pour pouvoir faire du développement web. De plus, le développeur web est très sollicité par les entreprises mais encore mieux il peut travailler indépendamment en se mettant a son propre compte.
